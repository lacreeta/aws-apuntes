# Amazon QuickSight

Servicio de **Business Intelligence (BI)** nativo de AWS para crear visualizaciones y dashboards interactivos.

Equivalente en Azure: **Power BI** (no idéntico, pero similar en propósito).

---

## Qué es Amazon QuickSight

- Servicio totalmente administrado de BI.
- Permite análisis ad-hoc y dashboards interactivos.
- Diseñado para escalar de forma automática.
- Se integra con múltiples fuentes de datos AWS.

---

## Casos de uso

- Visualización de datos empresariales.
- Análisis de costes y métricas.
- Dashboards para usuarios de negocio.
- Reporting en la nube.

---

## Claves de examen

- Servicio de Business Intelligence.
- Totalmente administrado.
- Alternativa BI nativa de AWS.
- No es tan completo como Power BI, pero cumple la misma función.
