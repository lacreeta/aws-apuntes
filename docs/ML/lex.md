# Amazon Lex

Amazon Lex es un servicio para crear **chatbots e interfaces conversacionales**.

---

## Qué problema resuelve
Permite interactuar con usuarios mediante **lenguaje natural**, texto o voz.

---

## Qué hace Lex
- Procesa lenguaje natural.
- Gestiona intenciones (intents).
- Integra texto y voz.
- Base de Amazon Alexa.

---

## Características clave
- AI preentrenado.
- Integración con Lambda.
- Multicanal (chat, voz).

---

## Casos de uso
- Chatbots de atención al cliente.
- Asistentes virtuales.
- Interfaces conversacionales.

---

## Pistas de examen
- Chatbot → Lex.
- Conversational AI.
- Integración con Lambda.

---

## Resumen rápido
Lex = **chatbots**
