# Amazon SageMaker

Amazon SageMaker es el servicio de AWS para **crear, entrenar y desplegar modelos de Machine Learning** de forma totalmente gestionada.

---

## Qué problema resuelve
Permite desarrollar modelos de ML **a medida** sin gestionar infraestructura compleja.

Es el servicio central de ML en AWS.

---

## Qué hace SageMaker
- Preparación de datos.
- Entrenamiento de modelos.
- Despliegue de endpoints.
- Gestión completa del ciclo de vida de ML.

---

## Características clave
- Servicio totalmente gestionado.
- Soporta frameworks populares (TensorFlow, PyTorch, etc.).
- Escalable.
- Integración con otros servicios AWS.

---

## Casos de uso
- Modelos de ML personalizados.
- Predicciones a medida.
- Proyectos de data science.
- Casos donde no sirve un modelo preentrenado.

---

## Pistas de examen
- ML **a medida** → SageMaker.
- “Build, train and deploy ML models”.
- No es AI preentrenado.

---

## Resumen rápido
SageMaker = **Machine Learning personalizado**
