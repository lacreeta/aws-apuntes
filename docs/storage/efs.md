# Amazon EFS (Elastic File System)

Amazon EFS es un servicio de **almacenamiento de archivos** totalmente gestionado que puede ser compartido entre múltiples instancias EC2.

## Qué problema resuelve
Permite que varias instancias EC2 accedan al mismo sistema de archivos simultáneamente.

## Características clave
- File system compartido
- Escala automáticamente
- Acceso concurrente desde múltiples EC2
- Alta disponibilidad (multi-AZ)

## Casos de uso
- Aplicaciones compartidas
- Contenidos web
- Home directories
- Workloads Linux

## Pistas de examen
- File system compartido
- Múltiples EC2 acceden al mismo storage
- No es block storage

## Resumen rápido
EFS = **file system compartido para EC2**
