# Amazon EBS (Elastic Block Store)

Amazon EBS es un servicio de **almacenamiento en bloques** diseñado para usarse con **instancias EC2**.

## Qué problema resuelve
Proporciona almacenamiento persistente para instancias EC2, similar a un disco duro virtual.

## Características clave
- Almacenamiento en bloques
- Se adjunta a instancias EC2
- Persistente (los datos sobreviven al reinicio)
- Alto rendimiento y baja latencia

## Casos de uso
- Sistemas operativos de EC2
- Bases de datos en EC2
- Aplicaciones que requieren acceso rápido a disco

## Pistas de examen
- EBS = disco duro para EC2
- No es serverless
- Está ligado a una Availability Zone

## Resumen rápido
EBS = **disco en bloques para EC2**
