# AWS Glue

AWS Glue es un servicio **serverless de ETL** (Extract, Transform, Load) para preparar datos para analytics.

## Qué problema resuelve
Automatiza la preparación y transformación de datos para análisis.

## Características clave
- Serverless
- ETL totalmente gestionado
- Glue Data Catalog integrado
- Integración con S3, Athena y Redshift

## Casos de uso
- Limpieza de datos
- Transformación de datasets
- Construcción de data lakes
- Preparación de datos para analytics

## Pistas de examen
- ETL → Glue
- Data Catalog
- No se gestionan servidores

## Resumen rápido
Glue = **ETL serverless**
