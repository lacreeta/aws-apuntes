# Amazon MSK (Managed Streaming for Apache Kafka)

Amazon MSK es un servicio **gestionado** que permite ejecutar **Apache Kafka** en AWS para el procesamiento de datos en tiempo real.

## Qué problema resuelve
Elimina la necesidad de instalar y mantener Kafka manualmente en EC2.

## Características clave
- Kafka gestionado por AWS
- Usa brokers sobre EC2
- Escalado y configuración manual
- Alta disponibilidad integrada

## Casos de uso
- Streaming de eventos
- Logs y métricas
- Integración entre microservicios
- Pipelines de datos

## Pistas de examen
- Apache Kafka → MSK
- No es serverless
- Requiere gestionar capacidad

## Resumen rápido
MSK = **Kafka gestionado (no serverless)**
