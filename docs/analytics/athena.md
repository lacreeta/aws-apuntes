# Amazon Athena

Amazon Athena es un servicio **serverless** que permite **consultar datos en Amazon S3 usando SQL**, sin necesidad de servidores.

## Qué problema resuelve
Permite analizar datos directamente en S3 sin moverlos ni crear bases de datos.

## Características clave
- Serverless
- Usa SQL estándar
- No requiere infraestructura
- Pago por consulta ejecutada

## Casos de uso
- Análisis ad-hoc
- Consultas sobre logs
- Data lakes en S3
- Business Intelligence

## Pistas de examen
- SQL sobre S3
- No hay bases de datos
- Pago por query

## Resumen rápido
Athena = **SQL serverless sobre S3**
