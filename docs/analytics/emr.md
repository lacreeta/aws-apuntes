# Amazon EMR (Elastic MapReduce)

Amazon EMR es un servicio **gestionado** para procesar grandes volúmenes de datos usando frameworks como **Apache Spark y Hadoop**.

## Qué problema resuelve
Facilita la ejecución de workloads de big data sin instalar frameworks manualmente.

## Características clave
- Usa instancias EC2
- Requiere gestionar clústeres
- Escalado manual
- Pago por EC2 + EMR

## Casos de uso
- Big data processing
- Batch analytics
- Machine learning a gran escala

## Pistas de examen
- Spark / Hadoop
- Clústeres
- No es serverless

## Resumen rápido
EMR = **Big Data gestionado sobre EC2**
