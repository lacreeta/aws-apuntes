# AWS Budgets

AWS Budgets permite crear **presupuestos y alertas** basadas en costes o uso.

---

## Qué problema resuelve
Ayuda a **evitar sobrecostes** avisando cuando se superan umbrales definidos.

---

## Características clave
- Presupuestos por:
  - Coste
  - Uso
  - RI / Savings Plans
- Alertas por email.
- Seguimiento proactivo.

---

## Casos de uso
- Controlar gasto mensual.
- Alertas antes de exceder un presupuesto.
- Control financiero proactivo.

---

## Pistas de examen
- Budgets = **alertas**.
- No muestra gráficos detallados.
- Se usa junto con Cost Explorer.

---

## Resumen rápido
Budgets = **presupuestos con alertas**
