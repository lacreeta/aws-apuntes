# AWS Cost Explorer

AWS Cost Explorer es una herramienta para **analizar y visualizar el gasto real** de AWS.

---

## Qué problema resuelve
Permite entender **en qué se gasta el dinero** y detectar patrones de uso.

---

## Características clave
- Gráficos interactivos.
- Análisis histórico.
- Filtros por:
  - Servicio
  - Cuenta
  - Región
  - Etiquetas
- Predicción de gasto futuro (basada en histórico).

---

## Casos de uso
- Analizar costes por servicio.
- Detectar picos de gasto.
- Optimización de costes.

---

## Pistas de examen
- Cost Explorer = **análisis histórico**.
- Muestra costes reales.
- No sirve para poner límites ni alertas.

---

## Resumen rápido
Cost Explorer = **analizar gasto real**
