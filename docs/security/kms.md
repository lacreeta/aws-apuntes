# AWS Key Management Service (KMS)

AWS KMS es un servicio **gestionado** para crear y gestionar **claves de cifrado** y controlar su uso.

## Qué problema resuelve
Centraliza la gestión de claves para cifrar datos en servicios de AWS.

## Características clave
- Gestión centralizada de claves
- Integración con servicios AWS
- Control de acceso mediante IAM
- Auditoría con CloudTrail

## Casos de uso
- Cifrado de datos en S3
- Cifrado de EBS y RDS
- Gestión de claves de seguridad

## Pistas de examen
- Encryption at rest
- Claves gestionadas por AWS
- Integración con IAM

## Resumen rápido
KMS = **gestión de claves de cifrado**
