# AWS Audit Manager

AWS Audit Manager es un servicio que ayuda a **recopilar y organizar evidencias** para auditorías de cumplimiento.

---

## Qué problema resuelve
Reduce el esfuerzo manual necesario para **preparar auditorías** de seguridad y compliance.

---

## Qué hace Audit Manager
- Recopila evidencias automáticamente.
- Se integra con servicios AWS.
- Ayuda a cumplir estándares (ISO, SOC, PCI, etc.).

---

## Qué NO hace
- ❌ No audita AWS.
- ❌ No sustituye a auditorías externas.
- ❌ No protege recursos.

---

## Casos de uso
- Preparación de auditorías.
- Cumplimiento normativo.
- Reducción de trabajo manual en compliance.

---

## Pistas de examen
- Evidence collection.
- Audit preparation.
- Compliance automation.

Si hablan de **evidencias** → Audit Manager.

---

## Resumen rápido
Audit Manager = **evidencias para auditorías**
