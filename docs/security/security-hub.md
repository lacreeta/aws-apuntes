# AWS Security Hub

AWS Security Hub es un servicio que **centraliza y correlaciona findings de seguridad** de múltiples servicios de AWS y partners.

## Qué problema resuelve
Proporciona una **vista unificada de la postura de seguridad** de una cuenta o de múltiples cuentas de AWS.

## Qué hace
- Agrega alertas (findings) de seguridad
- Evalúa cumplimiento con estándares
- Muestra el estado general de seguridad

## Servicios que integra
- Amazon GuardDuty
- AWS Inspector
- IAM Access Analyzer
- AWS Firewall Manager

## Estándares soportados
- AWS Best Practices
- CIS Benchmarks
- PCI DSS

## Casos de uso
- Monitorización de seguridad
- Centralización de alertas
- Evaluación de compliance

## Pistas de examen
- Centralized security dashboard
- Aggregated findings
- Security posture

## Qué NO es
- No bloquea ataques
- No es un firewall
- No es un servicio de cifrado

## Resumen rápido
AWS Security Hub = **panel central de seguridad**
