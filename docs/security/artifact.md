# AWS Artifact

AWS Artifact es un servicio que proporciona **acceso bajo demanda a informes de cumplimiento y documentos de auditoría de AWS**.

## Qué problema resuelve
Permite a las empresas obtener fácilmente documentación oficial de AWS para cumplir requisitos de **compliance, auditoría y regulaciones**.

## Qué ofrece
- Informes de cumplimiento (SOC, ISO, PCI, etc.)
- Documentación legal y acuerdos
- Acceso bajo demanda desde la consola

## Casos de uso
- Auditorías de seguridad
- Requisitos regulatorios
- Compliance corporativo

## Pistas de examen
- Compliance reports
- Audit documents
- Certifications
- Legal documentation

## Qué NO es
- No protege contra ataques
- No cifra datos
- No detecta amenazas

## Resumen rápido
AWS Artifact = **documentos de compliance y auditoría**
