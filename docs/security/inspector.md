# AWS Inspector

AWS Inspector es un servicio que **evalúa automáticamente vulnerabilidades y exposiciones de seguridad** en workloads de AWS.

## Qué problema resuelve
Detecta **vulnerabilidades, malas configuraciones y riesgos de seguridad** en instancias y contenedores.

## Qué analiza
- Instancias EC2
- Imágenes de contenedores (ECR)
- Workloads en ejecución

## Qué detecta
- Vulnerabilidades de software (CVEs)
- Configuraciones inseguras
- Exposición a riesgos conocidos

## Casos de uso
- Escaneo de seguridad
- Gestión de vulnerabilidades
- Mejora de la postura de seguridad

## Pistas de examen
- Vulnerability assessment
- Security scanning
- CVEs

## Qué NO es
- No investiga incidentes
- No bloquea ataques
- No es compliance documentation

## Resumen rápido
AWS Inspector = **detección de vulnerabilidades**
