# Amazon Macie

Servicio de **descubrimiento y protección de datos sensibles** usando machine learning.

Equivalente en Azure: **DLP (Microsoft Purview)**

---

## Qué hace Macie

- Identifica datos sensibles (PII).
- Analiza principalmente buckets S3.
- Ayuda al cumplimiento normativo.

---

## Casos de uso

- Protección de datos personales.
- Auditorías de seguridad.
- Cumplimiento GDPR.

---

## Claves de examen

- Detecta PII.
- Usa machine learning.
- Orientado a S3.
