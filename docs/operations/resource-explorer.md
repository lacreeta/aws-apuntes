# AWS Resource Explorer

Servicio para **buscar y explorar recursos AWS** en una o varias cuentas.

Equivalente en Azure: **Azure Resource Graph Explorer**

---

## Qué hace Resource Explorer

- Permite buscar recursos por nombre, tipo o etiquetas.
- Ofrece visibilidad centralizada.
- Funciona a nivel de cuenta y organización.

---

## Casos de uso

- Localizar recursos rápidamente.
- Auditoría y gobernanza.
- Gestión de entornos grandes.

---

## Claves de examen

- Servicio de visibilidad.
- No crea ni modifica recursos.
- Orientado a gobernanza.
