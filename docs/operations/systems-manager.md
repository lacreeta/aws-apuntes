# AWS Systems Manager

Servicio para la **operación y administración de instancias y aplicaciones** en AWS.

Equivalente en Azure: **Azure Automation + Azure Arc**

---

## Qué hace Systems Manager

- Administración centralizada de instancias EC2 y on-premises.
- Ejecución remota de comandos.
- Gestión de parches y configuración.
- Inventario de recursos.

---

## Funcionalidades principales

- Run Command
- Patch Manager
- Automation
- Inventory
- Parameter Store

---

## Casos de uso

- Aplicar parches sin acceso SSH/RDP.
- Automatizar tareas operativas.
- Gestionar instancias a escala.

---

## Claves de examen

- Servicio de **operaciones (day-2)**.
- No requiere agentes manuales.
- Centraliza la gestión de instancias.
