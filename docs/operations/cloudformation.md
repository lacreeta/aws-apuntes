# AWS CloudFormation

Servicio de **Infraestructura como Código (IaC)** para definir y desplegar recursos AWS mediante plantillas.

Equivalente en Azure: **Azure ARM / Bicep**

---

## Qué hace CloudFormation

- Define recursos en plantillas YAML o JSON.
- Automatiza la creación, actualización y eliminación de recursos.
- Gestiona dependencias y orden de despliegue.
- Detecta cambios manuales (drift detection).

---

## Casos de uso

- Despliegue repetible de infraestructura.
- Automatización de entornos (dev, test, prod).
- Base para Service Catalog.

---

## Claves de examen

- Es IaC.
- Usa plantillas.
- Automatiza despliegues.
- No es un servicio de ejecución.
