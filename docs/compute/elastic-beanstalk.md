## AWS Elastic Beanstalk

AWS Elastic Beanstalk es un servicio **PaaS** que permite desplegar aplicaciones sin gestionar la infraestructura subyacente.

El usuario se centra en el **código**, AWS gestiona la infraestructura.

Equivalente en Azure: **Azure App Service**

---

## Características principales de Elastic Beanstalk

- Soporta múltiples lenguajes:
  - Java, .NET, Node.js, Python, PHP, Go, Docker.
- Despliegue automático de:
  - EC2
  - Auto Scaling
  - Load Balancer
- Monitorización integrada con CloudWatch.

---

## Modelo de responsabilidad en Elastic Beanstalk

- El usuario gestiona:
  - Código
  - Configuración de la aplicación
- AWS gestiona:
  - Servidores
  - Escalado
  - Balanceo
  - Parches del SO

---

## Casos de uso Elastic Beanstalk

- Aplicaciones web.
- APIs.
- Despliegues rápidos sin complejidad.

---

## Claves de examen Elastic Beanstalk

- Es PaaS.
- No gestionas servidores directamente.
- Ideal para desplegar rápido.
- Pagas solo por los recursos subyacentes.
