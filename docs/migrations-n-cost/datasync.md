# AWS DataSync

AWS DataSync permite migrar **grandes volúmenes de archivos** de forma rápida y automatizada.

## Qué problema resuelve
Acelera la transferencia de datasets basados en **file systems** hacia AWS.

## Qué migra
- Archivos
- Directorios
- Sistemas NFS / SMB

## Destinos típicos
- Amazon S3
- Amazon EFS
- Amazon FSx

## Casos de uso
- Migrar file servers
- Backups a AWS
- Sincronización híbrida

## Pistas de examen
- Large dataset of files
- File system
- NFS / SMB
- High-speed transfer

## No usar cuando
- Es una base de datos
- Necesitas CDC

## Resumen rápido
DataSync = **migración de archivos**
