# AWS Database Migration Service (DMS)

AWS DMS permite migrar **bases de datos** a AWS de forma segura y con **mínimo downtime**.

## Qué problema resuelve
Permite mover grandes bases de datos sin interrumpir aplicaciones en producción.

## Qué migra
- Datos estructurados
- Tablas y esquemas
- Cambios continuos (CDC)

## Orígenes y destinos
- On-premises → AWS
- EC2 → RDS / Aurora / DynamoDB
- Migraciones heterogéneas

## Casos de uso
- Migrar bases de datos grandes
- Migraciones con alta disponibilidad
- Modernización de bases de datos

## Pistas de examen
- Large database
- Minimal downtime
- Continuous replication
- Structured data

## No usar cuando
- Son archivos
- Es una aplicación completa

## Resumen rápido
DMS = **migración de bases de datos**
