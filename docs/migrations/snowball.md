# AWS Snowball

AWS Snowball es un servicio para migrar **enormes volúmenes de datos offline** usando dispositivos físicos.

## Qué problema resuelve
Permite migrar datos cuando la red es lenta, cara o no viable.

## Qué migra
- Datasets muy grandes
- Petabytes de información

## Casos de uso
- Migraciones iniciales masivas
- Conectividad limitada
- Centros de datos on-premises

## Pistas de examen
- Offline migration
- Petabytes of data
- Limited bandwidth

## No usar cuando
- La red es suficiente
- Los datos son pequeños

## Resumen rápido
Snowball = **migración física offline**
