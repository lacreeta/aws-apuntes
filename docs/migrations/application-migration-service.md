# AWS Application Migration Service

AWS Application Migration Service permite migrar **aplicaciones y servidores completos** a AWS con pocos cambios.

## Qué problema resuelve
Facilita migraciones tipo **lift-and-shift** hacia Amazon EC2.

## Qué migra
- Sistemas operativos
- Configuración
- Discos completos

## Casos de uso
- Migrar aplicaciones legacy
- Rehosting de servidores
- Migraciones rápidas sin rediseño

## Pistas de examen
- Lift and shift
- Minimal changes
- Rehost applications

## No usar cuando
- Solo quieres mover datos
- Solo migras una base de datos

## Resumen rápido
Application Migration Service = **migración de servidores**
