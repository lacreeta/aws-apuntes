# Amazon CloudFront

Amazon CloudFront es un servicio **CDN** para distribuir contenido con baja latencia.

---

## Qué problema resuelve
Reduce latencia y carga en los orígenes distribuyendo contenido desde **Edge Locations**.

---

## Características clave
- CDN global.
- Usa Edge Locations.
- Integración con S3, ALB y EC2.
- Mejora rendimiento y seguridad.

---

## Casos de uso
- Contenido estático.
- Streaming.
- Sitios web globales.

---

## Pistas de examen
- CloudFront ≠ Load Balancer.
- Usa Edge Locations.
- Mejora latencia global.

---

## Resumen rápido
CloudFront = **CDN de AWS**
