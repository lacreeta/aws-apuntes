# AWS Global Accelerator

AWS Global Accelerator mejora la **latencia y disponibilidad** del tráfico global.

---

## Qué problema resuelve
Optimiza el routing global usando la red de AWS.

---

## Características clave
- IPs estáticas globales.
- Enrutamiento optimizado.
- Mejora disponibilidad.

---

## Casos de uso
- Aplicaciones globales.
- Usuarios distribuidos.
- Alta disponibilidad.

---

## Pistas de examen
- No es CDN.
- Optimiza tráfico TCP/UDP.
- Usa red global de AWS.

---

## Resumen rápido
Global Accelerator = **optimización de tráfico global**
