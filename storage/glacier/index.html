    Amazon S3 Glacier¶

    Amazon S3 Glacier es una clase de almacenamiento de Amazon S3 diseñada para archivado a largo plazo y coste muy bajo.

    Qué problema resuelve¶

    Permite almacenar datos que se usan muy poco, pero que deben conservarse durante años de forma segura y económica.

    Características clave¶

    • Coste muy bajo
    • Alta durabilidad
    • Acceso no inmediato (minutos u horas)
    • Integrado dentro de Amazon S3

    Clases de S3 Glacier¶

    • Glacier Instant Retrieval → acceso inmediato (poco uso)
    • Glacier Flexible Retrieval → minutos u horas
    • Glacier Deep Archive → horas (el más barato)

    Casos de uso¶

    • Backups antiguos
    • Datos históricos
    • Cumplimiento legal
    • Archivado a largo plazo

    Pistas de examen¶

    • Datos a largo plazo
    • Acceso poco frecuente
    • Coste mínimo
    • No se usa para acceso en tiempo real

    Resumen rápido¶

    Glacier = archivado barato a largo plazo

    Checklist para el Examen

    • Entiendo qué es este servicio
    • Conozco sus casos de uso principales
    • Sé cuándo NO usarlo
    • Comprendo el modelo de precios básico
