    Amazon Detective

    Amazon Detective es un servicio que analiza y correlaciona datos de seguridad para investigar y entender incidentes de seguridad.

    Qué problema resuelve

    Facilita la investigación de incidentes mostrando cómo y por qué ocurrió un evento de seguridad.

    Qué hace

    • Analiza datos de seguridad automáticamente
    • Correlaciona eventos de múltiples fuentes
    • Ayuda a investigar comportamientos sospechosos

    Datos que usa

    • AWS CloudTrail
    • Amazon VPC Flow Logs
    • Amazon GuardDuty findings

    Casos de uso

    • Investigación de incidentes
    • Análisis de causa raíz
    • Comprensión de ataques o accesos sospechosos

    Pistas de examen

    • Investigate security issues
    • Analyze security events
    • Root cause analysis

    Qué NO es

    • No previene ataques
    • No bloquea tráfico
    • No es un firewall

    Resumen rápido

    Amazon Detective = investigar y entender incidentes de seguridad
