    Amazon Kinesis Data Streams¶

    Amazon Kinesis Data Streams es un servicio serverless de AWS para la ingestión y procesamiento de datos en tiempo real.

    Qué problema resuelve¶

    Permite capturar grandes volúmenes de datos de múltiples productores y procesarlos con baja latencia.

    Características clave¶

    • Serverless (no se gestionan servidores)
    • Procesamiento en tiempo real
    • Alta escalabilidad
    • Integración nativa con servicios AWS

    Casos de uso¶

    • Real-time analytics
    • Streaming de logs
    • Datos de IoT
    • Clickstreams

    Pistas de examen¶

    • Streaming en tiempo real
    • Alternativa nativa AWS a Kafka
    • No requiere gestionar brokers

    Resumen rápido¶

    Kinesis Data Streams = streaming serverless en AWS
